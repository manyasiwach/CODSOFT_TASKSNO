# Task 1 - Data Cleaning & Preprocessing

## Objective
The objective of this task is to clean and preprocess the Iris dataset by inspecting its structure, checking for missing values, removing duplicate records, verifying data types, and preparing a clean dataset for further analysis.

## Dataset
- Iris Dataset (Scikit-learn)

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Steps Performed
- Imported the dataset
- Inspected dataset structure
- Checked missing values
- Checked duplicate records
- Removed duplicate records
- Verified data types
- Saved the cleaned dataset as `iris_cleaned.csv`

## Files
- Task_1_Data_Cleaning_Preprocessing.ipynb
- iris_cleaned.csv

## Outcome
The dataset was successfully cleaned and prepared for further analysis and machine learning tasks.
